一、积分使用规则

（一）、使用顺序

勾选支付时，在计算订单金额的时候，计算支付的先后顺序：

1. 先算行销活动折抵金额/买立减金额

2. 再算优惠券/购物金/抵用券

3. 再算积分折抵金额

4. 再算购物卡

5. 最后算其他的支付方式

（二）、使用限制

1. 商品的单价×数量 >= 5元

2. 积分不可抵扣运费

（三）、使用范围

目前自营、转单、门店、的商品都可以使用/累计积分，商城、预购商品不可用。除部分特殊商品不可用积分抵扣，可在PM平台通过设定分类和卖场来排除

（四）、使用效期

今年获得的积分有效期至次年12/31，不可自主设定

（五）、获得规则

1.商品的单价×数量 >= 5元时： 商品A可获得积分 = 商品A支付金额×积分返还比例×100，商品A支付金额 = 商品A小计金额 - 商品A参加行销活动折抵的金额 - 优惠券/购物金/抵用券抵扣分摊后金额 - 积分折抵分摊后金额；订单中含多个商品时，订单可获得积分等于多个商品可获得积分的累加；部分特殊商品不可获得积分

2.计算出的积分若有小数部分，则无条件进位。即10.2-->11

3.商品的单价×数量 < 5元时，不累计积分

（六）、抵扣规则

1.商品的单价×数量 >= 5元时： 商品A可获得积分 = 商品A支付金额×积分返还比例×100，商品A支付金额 = 商品A小计金额 - 商品A参加行销活动折抵的金额 - 优惠券/购物金/抵用券抵扣分摊后金额 - 积分折抵分摊后金额；订单中含多个商品时，订单可获得积分等于多个商品可获得积分的累加；部分特殊商品不可获得积分

2.计算出的积分若有小数部分，则无条件进位。即10.2-->11

3.商品的单价×数量 < 5元时，不累计积分

（七）、退订规则

1.用户取消订单或退货时，优先返还到期日较晚的积分，积分有效期不变

2.用户取消订单时，回收用户的待生效积分

3.用户退货时：

如果积分在待生效期，系统回收待生效积分

如果积分已转为可用，但是客人未将积分使用，系统会回收积分

如果积分已转为可用，且积分已被客人使用，系统不回收积分

4.回收积分、退积分与退款的时间节点一致并返还至原账户

二、分摊规则

（一）、分摊计算顺序

1.买立减/行销活动

2.优惠券/抵用券/购物金/活动用购物卡

3.积分

4.购物卡

（二）、分摊算法

商品为可积分抵扣商品时：

商品积分分摊 =（商品折扣后金额 / 订单中可积分抵扣商品的总折扣后金额）× 该订单使用的积分

商品积分优惠金额分摊 = 商品积分分摊 × 0.01元

商品折扣后金额 = 商品小计金额 - 商品参加行销活动折抵的金额 - 优惠券/购物金/抵用券抵扣分摊后金额

分摊后的积分若有小数部分，采用四舍五入；最后一个分摊的积分 = 总共使用的积分 - 之前已分摊的积分

案例1：

订单1中仅有一个商品A售价50元，参加了行销活动满50减5（封顶），使用了一张抵用券，使用了1000积分：

则商品A小计金额=50；行销活动折抵金额=5；抵用券抵扣分摊后金额=45×20%=9；

商品A折扣后金额 = 50-5-9

商品A积分分摊 = 1×1000 = 1000

商品积分优惠金额分摊 = 1000×0.01元 = 10元

案例2：

订单2中含有商品A 2件，单价50元，参加了行销活动满50减5（封顶），使用了一张抵用券，使用了1000积分：

则商品A小计金额 = 100；行销活动折抵金额 = 5；抵用券抵扣分摊后金额 = 95×20%=19；

商品A折扣后金额 = （100-5-19）/2

商品A积分分摊 = 0.5×1000 = 500

商品A积分优惠金额分摊 = 500×0.01元 = 5元

案例3：

订单3中含有两个商品，商品A单价30元和商品B单价90元，A与B参加了行销活动满100减10，使用了一张抵用券，使用了1000积分：

则商品A小计金额 = 30，行销活动折抵金额 = 10×30/(30+90)=2.5；

商品B小计金额 = 90，行销活动折抵金额 = 10×90/(30+90)=7.5；

商品A抵用券抵扣分摊后金额 = (120-7.5-2.5)×20%×(30-2.5)/(30-2.5+90-7.5)=5.5；

商品B抵用券抵扣分摊后金额 = (120-7.5-2.5)×20%×(90-7.5)/(30-2.5+90-7.5)=16.5；

商品A折扣后金额 = 30-2.5-5.5=22；商品B折扣后金额 = 90-7.5-16.5=66

商品A积分分摊 = 22/88×1000 = 250；商品B积分分摊 = 总共使用的积分 - 商品A积分分摊=1000-250=750

商品A积分优惠金额分摊 = 250×0.01元 = 2.5元；商品B积分优惠金额分摊 = 750×0.01元 = 7.5元

案例4：

订单4中含有三个商品，商品A单价30元，商品B单价90元，商品C单价70元，A与B参加了行销活动满100减10，B和C满足优惠券满100减15，使用了1000积分：

则商品A小计金额 = 30；商品B小计金额 = 90；商品C小计金额 = 70

商品A行销活动折抵金额 = 10×30/(30+90)=2.5；商品B行销活动折抵金额 = 10×90/(30+90)=7.5

商品B优惠券抵扣分摊后金额 = 15×(90-7.5)/（90-7.5+70）=8.14；商品C优惠券抵扣分摊后金额 = 15-8.14=6.86

商品A折扣后金额 = 30-2.5=27.5；商品B折扣后金额 = 90-7.5-8.14=74.36；商品B折扣后金额 = 70-6.86=63.14

商品A积分分摊 = 27.5/（27.5+74.36+63.14）×1000 ；商品B积分分摊 = 74.36/（27.5+74.36+63.14）×1000；商品C积分分摊 = 1000 - 商品A积分分摊 - 商品B积分分摊
